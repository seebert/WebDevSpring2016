about the assignemnt

The assignment application, called Form Maker, was built throughout the semester in a series of biweekly assignments. It consists of a Website that allows users to create forms and share them on the Web as single pages or embedded in their own Web pages. We will refer to those users creating forms as developers. Other users that visit the Web pages or form pages can then fill in and submit the forms. We will refer to these as submitters.

about the project

The project is a web application that can be used by groups to track money owed to one another, whether from monthly bills, group events, or other miscellaneous spending. Users that register with the application can create new events, designating themselves as manager. Within the new event page, the manager can create subcategories of charges and invite other individuals to complete payments. The manager is granted permissions to track the payments and view which individuals have successfully repaid expenses.

This application was inspired by a ski trip I attended earlier in the semester. One individual fronted the money for the cost of the house, another person groceries, and several others gas. This application would have allowed the trip organizer to create a single event page (i.e. “Ski trip”) and create detailed expense categories (i.e. house, groceries, gas) that outline which individuals owe money and to whom. Ideally, I will be able to incorporate a PayPal API to complete charges from within the application.
